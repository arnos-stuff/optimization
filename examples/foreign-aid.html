Consider the following tweet:

you can choose a country, or countries, composed of no more than 150m people, to immediately increase in gdp/c by $10,000. which do you choose and why

— no brain ((_night_brain__?)) June 7, 2023

This, if we denote the GDP as the variable g \geq 0, and the population as p \geq 0, turns into the min-max problem:

\begin{align*} &\min_g\left\lbrace\max_p\left\lbrace\dfrac{g}{p}\right\rbrace\right\rbrace\\ &\text{such that}\\ &\qquad\qquad p \leq P_T \\ &\qquad\qquad g \geq 0\\ &\qquad\qquad p \geq 0\\ \end{align*}

where P_T = 150\cdot 10^6 is an upper bound on the population level.

4.1 The log trick

as \log(\cdot) is increasing, convex and smooth, any optimization problem can be taken to log space without affecting the objective.

this yields the objective \tilde{C}(\theta) \triangleq \log g - \log p, with \theta = (p,g).

Further, as we know that \min_z f(z) = -\max_z\{-f(z)\} are attained at the same point z^\star, we can reframe the min-max problem as:

\min_{\theta}\left\lbrace C(\theta)\right\rbrace \triangleq \min_{\theta}\left\lbrace \log g + \log p\right\rbrace = \min_{\theta}\left\lbrace \mathbf{1}^\mathsf{T}\theta\right\rbrace

the constraints become \theta_1, \theta_2 \in \mathbb{R} and \theta_1 \leq \log P_T.

4.2 But what is the domain ?

We haven’t really defined constraints on g and p, they can’t obviously be anything.

As the problem states here, the goal is to be any finite combination of states. Let N(P_T) \triangleq \lvert \mathbf{C}(P_T) \rvert \triangleq\lvert \{ i \in \mathbb{N} \mid g_i \leq P_T \}\rvert

This is the set of countries whose GDP is less than P_T.

Let us figure out how many they are.

Code 
class Country:
  data = dfs
  @staticmethod
  def underPop(threshold: int):
    return dfs.loc[dfs.population < threshold, 'country.value'].values.tolist()
  @staticmethod
  def underPopCount(threshold: int):
    return len(dfs.loc[dfs.population < threshold, 'country.value'].values.tolist())
  @staticmethod
  def underPopMatrix(threshold: int):
    return dfs.loc[dfs.population < threshold, ['gdp', 'population']].values

  @staticmethod
  def underPopDf(threshold: int):
    return dfs.loc[dfs.population < threshold].rename({'country.value':'country'})

print(f"The number of countries with population less than {150} million is: {Country.underPopCount(150*10**6)}")

4.3 Convex Hull formulation

So we are going to set N = N(P_T) = 183. The goal is hence to find any point \mathbf{x}\in\mathbf{Hull}(\Theta) with \Theta = \{\theta_i\}_{i\in\mathbf{C}(P_T)} so that \mathbf{1}^\mathsf{T}\mathbf{x} is minimal.

This means we are instead optimizing on the weights we give to each country w_i \in [0,1], this weight corresponds to the fraction of the country that is going to receive aid.

Let \mathbf{w}\in [0,1]^N be those weights, one per country. The problem can be cast into a LP through the constraints

\mathbf{1}^\mathsf{T}\mathbf{w} = 1 \quad\text{and}\quad \mathbf{C}\mathbf{w} = \mathbf{x} \quad\text{and}\quad \mathbf{w}\geq 0

where

\mathbf{C} = \begin{bmatrix} \theta^1 & \theta^2 & \cdots &\theta^i & \cdots & \theta^N \end{bmatrix} = \begin{bmatrix}\theta_1^1 & \theta_1^2 & \cdots &\theta_1^i & \cdots & \theta^N_1 \\ \theta_2^1 & \theta_2^2 & \cdots &\theta_2^i & \cdots & \theta^N_1 \end{bmatrix}

Hence

is just a matrix where we store the N possible countries’ GDP and populations.

This is not yet a LP, unless you realize you can just recast the objective as \mathbf{1}^\mathsf{T}\mathbf{C}\mathbf{w} \triangleq \mathbf{c}^\mathsf{T}\mathbf{w}.

As per the constraint, for the upper bound we just need that \mathbf{B}\mathbf{C}\mathbf{w} = \mathbf{B}\mathbf{x}\leq \mathbf{b} where \mathbf{B} is the matrix

\mathbf{B} = \left[\begin{array}{c:c} 0 & 0\\\hdashline 0 & 1\end{array}\right]

And the constraint \mathbf{1}^\mathsf{T}\mathbf{w} = 1 can be transformed into the variable integrates into the final block matrix

\mathbf{A} = \left[\begin{array}{c} \mathbf{1}^\mathsf{T}\\ \hdashline \mathbf{B}\mathbf{C}\end{array}\right] \implies \mathbf{A}\mathbf{w} = \mathbf{b}\quad\text{with}\quad\mathbf{b}=\left[\begin{array}{c} 1 \\ 0\\P_T\end{array}\right]

4.3.1 Addendum

It is clear that the true constraint isn’t \mathbf{A}\mathbf{x} = \mathbf{b}, as one of the constraints is an inequality. This can be reality alleviated using so-called “slack variables”, but solvers usually do that for us so we won’t yet bother doing it manually.

4.4 Final statement

The final problem is

\begin{align*} &\min_\mathbf{w}\quad \mathbf{c}^\mathsf{T}\mathbf{w}\\ &\text{such that}\quad\mathbf{A}\mathbf{w} = \mathbf{b}\\ &\qquad\qquad\qquad\mathbf{w} \geq 0 \end{align*}

5 Solving the problem

We are going to use the nice library CVX to solve the problem.

Code 
import cvxpy as cp
import numpy as np

PopMax = 150*10**6
N = Country.underPopCount(PopMax)

C = Country.underPopMatrix(PopMax)
C = cp.Constant(np.log(C))
B = cp.Constant([[0,0],[0,1]])
b = cp.Constant(np.array([0, np.log(PopMax)]).reshape(-1,1))
ones = cp.Constant(np.ones((N,1)))
agg = cp.Constant(np.ones((2,1)))
c =  C @ agg 
w = cp.Variable(shape=(N,1), pos=True)

prob = cp.Problem( cp.Minimize( c.T @ w ),
constraints=[
    w >= 0,
    ones.T @ w == 1,
    B @ C.T @ w <= b]
    )

prob.solve(verbose=False)

dfr = Country.underPopDf(PopMax)
dfr['percentage'] = np.round(w.value,10)
dfr[dfr.percentage > 0].sort_values('percentage', ascending=False)
